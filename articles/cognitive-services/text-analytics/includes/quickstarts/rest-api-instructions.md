1. 将命令复制到文本编辑器中。
2. 必要时在命令中进行如下更改：
    1. 将值 `<your-text-analytics-key-here>` 替换为你的值。
    2. 将请求 URL 的第一部分 (`<your-text-analytics-endpoint-here>`) 替换为你自己的终结点 URL。
3. 打开命令提示符窗口。
4. 将文本编辑器中的命令粘贴到命令提示符窗口，然后运行命令。
