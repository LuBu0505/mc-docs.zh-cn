# <a name="access-the-admin-console"></a>访问管理控制台

若要访问管理控制台，请转到工作区 UI 的右上方，单击 ![“帐户”图标](../_static/images/icons/account-icon.png) 用户帐户图标，然后选择“管理控制台”。

只有 Azure Databricks [管理员](users-groups/users.md)可访问管理控制台。
